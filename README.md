# 🌐 Présentation - Session Réseaux & Déploiement Web


## 🎯 Objectif de la session
Cette session a pour objectif de :
- Comprendre les bases des réseaux informatiques
- Maîtriser l’environnement Linux (Ubuntu)
- Apprendre à déployer une application web
- Mettre en pratique via des labs et ateliers

---

## 📚 Phase 1 : Tutorials (Apprentissage)

Chaque membre a réalisé un tutoriel :

### 🔹 Adressage IP
- Comprendre IPv4 / IPv6
- Notion de réseau et sous-réseau

### 🔹 LAN & Équipements
- Définition d’un réseau local (LAN)
- Les équipements :
  - Routeur
  - Switch
  - Câbles réseau

### 🔹 Ubuntu (Linux)
- Installation de Ubuntu
- Commandes de base
- Gestion des fichiers et utilisateurs

---

## 🎨 Communication

### 📢 Création d’un Flyer
Un flyer a été conçu pour présenter les ateliers de fin de session :

- 🐧 Installation Linux  
- 🌐 LAN  
- 🚀 Déploiement Laravel  

Objectif : Attirer les apprenants et expliquer les ateliers

## 🚀 Phase 2 : Lancement des Activités

### 📝 Inscription des apprenants
- Mise en place d’un système d’inscription
- Choix des ateliers par les participants

---

## 🧪 Labs (Travaux Pratiques)

Des labs ont été réalisés pour renforcer la compréhension :

### 🔸 Installation & Utilisation Ubuntu
- Installation du système
- Utilisation du terminal

### 🔸 Configuration LAN
- Connexion entre machines
- Test de communication (ping)

### 🔸 Interconnexion Réseau
- Communication entre différents réseaux
- Configuration basique

---

## 🔥 Phase 3 : Ateliers

Organisation des ateliers pratiques :

### 🐧 Atelier Installation Linux
- Installation complète
- Configuration de base

### 🌐 Atelier LAN
- Création d’un réseau local
- Test de connectivité

### 🚀 Atelier Déploiement Laravel
- Mise en ligne d’une application
- Configuration serveur

---

## 📊 Résultats

- Acquisition de compétences pratiques
- Travail en équipe
- Compréhension réelle des concepts

- Passer de la théorie à la pratique
